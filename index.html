what I do

social media web app

Currently working on a social media web application for pet owners. This app will enable users to exchange ideas and find new ways to groom and breed their pets. The front-end of the app is build with a Typescript variant of React JS and styled with custom CSS. The back-end relies on Firebase-v9 for hosting, authentication, storing images and saving data.

react JS CRUD apps

A full stack CRUD application that includes authentication, hosting and data management via Firebase-v9 libraries. The styling was provided with the help of React-Bootstrap5. The app is packaged using ViteJS and also uses version 6 of React-Router-Dom.

nextJS blog/ app

A movie blog/app developed with Next.JS that had its specifications requested by a client. The app uses Contentful to store and manage data. React-Bootstrap5 was used for the styling. Another package of relevance was React-Player. It was used to enable a swift development of the media features.

my work

who am I?

Web developer based in London.

I have over ten years experience in web development. Initially build websites using content management systems such as Drupal 7 as an administrator. I then moved into CSS to customize the styling of Drupal websites. JavaScript became the next logical step in the web developer learning path. I was no stranger to programming before JavaScript as I was already reasonably versed in Java. This made the move to Typescript even more attractive. Today, I use Typescript for all my web endeavors, including on the server side with Firebase.

Prior to embarking on a full time web development career, I worked as a data analysis technician for a Reinsurance company in the city of London. This work included, but was not limited to, collecting and administering broker reinsurance files and preparing the files for broker use.
